- packer.nvim
- nvim >= 0.9
- luajit
- *a brain*

- `aurora` theme
- `bufferline.nvim` for top-panel with buffer list

- `neo-tree.nvim` 
- `gitsigns.nvim`
- `lualine.nvim`
- `nvim-web-devicons`
- `nvim-lspconfig`
- `mason-lspconfig`
